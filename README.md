# CourseMate Website

Marketing site for [CourseMate](https://apps.apple.com/us/app/coursemate/id6752738583) — the social golf scoring app. Built with Next.js (App Router), React and Tailwind CSS v4.

## Pages

- `/` — home page with features, Pro and download links
- `/terms` — Terms of Service
- `/privacy` — Privacy Policy

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on [Vercel](https://vercel.com). Push to the repository's default branch to trigger a production deploy — no extra configuration required (Vercel auto-detects Next.js).

## Design source

The original Claude Design export lives in `design/` — the site pages are a faithful Next.js implementation of those files.
